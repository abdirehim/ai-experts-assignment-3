# AI Experts Assignment (Python)

# Running Tests

### Locally

```bash
pip install -r requirements.txt
pytest -v
```

### With Docker

```bash
docker build -t ai-experts .
docker run ai-experts
```


## Optional: Development Environment

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
pytest -v
```
